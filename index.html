As an example of such crousework, for my Data Visualization class is a tool I created: Sub Group Explorer Tool in an effort to address a common issue in data science, which is finding interesting sub groups within a large dataset. It takes as input a dataset and outputs multivariate visualizations of sub groups. The tool can be used in an exploritory manner to look at the statistical distributions and correlations between data, as well as a visualization tool once the interesting relationships have been determined.
